# 实战计划生存法则

1. 编程是一个 learning by doing 的技能，当你能够完成这道练习题，你才真正学会了这节课的知识点；
2. 如果做练习的过程中遇到困难，善用搜索找答案；
3. 初学者都不爱看文档，可是文档很重要。所以我们的一些题目会引导你去看文档，当做字典去查阅就好；
4. 我们提供了练习题参考答案，不过最好在独立完成之后，再对照答案；
5. 需要按时提交作业，逼自己一把，deadline 是第一生产力。
